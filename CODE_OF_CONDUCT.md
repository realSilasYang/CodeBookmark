# 社区行为准则

## 承诺

CodeBookmark 社区致力于为所有参与者提供开放、友善、尊重且不受骚扰的协作环境，不因经验、身份、背景、能力或观点差异而区别对待。

## 期望行为

- 使用尊重、清晰且有建设性的语言讨论技术问题。
- 接受合理的技术质疑，并用事实、测试和可复现证据支持结论。
- 尊重他人的时间、隐私、署名和知识产权。
- 在发现错误时及时更正，在产生分歧时聚焦项目目标和实际影响。
- 对新参与者提供必要上下文，不进行人身评价。

## 不可接受的行为

- 骚扰、威胁、歧视、侮辱、跟踪或公开他人隐私信息。
- 针对个人而非技术内容的攻击，或持续进行破坏性争论。
- 发布垃圾信息、恶意代码、未经授权的密钥、书签数据或其他敏感内容。
- 任何在专业协作环境中明显不适当的行为。

## 适用范围

本准则适用于仓库、Issue、Pull Request、代码审查、Release 讨论以及代表本项目参与的其他公开或私密交流。

## 执行

发现违反本准则的行为时，请通过维护者 GitHub 主页提供的私密联系方式报告。没有其他私密渠道时，可以使用仓库的 Private vulnerability reporting，并在标题中注明“行为准则报告”。请勿在公开 Issue 中披露受影响人员的隐私。

维护者会根据事实和影响采取提醒、内容删除、临时限制或永久禁止参与等措施，并尽力保护报告者和受影响人员的隐私。维护者同样受本准则约束。
